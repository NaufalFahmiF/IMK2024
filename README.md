### Tutorial Clone
1. Clone project ini lewat "<> Code" lalu copy link git clonenya.
2. Buat direktori lalu buka git bash dan ketikkan
```bash
git clone <link clone project ini>
```
3. Buka folder
4. Buka terminal ketikkan
```shell
npm i
```
5. Lalu ketik
```shell
cd frontend
```
6. Kemudian ketikkan
```shell
npm i
```
